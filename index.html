옵시디언 핵심 개념

      

        

          
🗄️

          
Vault

          
노트가 모여 있는 폴더 하나가 곧 Vault예요. 옵시디언은 이 폴더 안의 파일을 그대로 읽고 쓰기 때문에, 내 노트는 언제나 내 컴퓨터에 남아요.

        

        

          
📝

          
노트

          
하나의 마크다운(.md) 파일이 하나의 노트예요. 하나의 노트에는 하나의 생각만 담는 것이 연결을 만들기 좋은 습관이에요.

        

        

          
🏷️

          
태그

          
#생산성처럼 #을 붙여 노트를 분류해요. 폴더와 달리 한 노트에 여러 태그를 달 수 있어 유연하게 묶을 수 있어요.

        

        

          
🔗

          
내부링크

          
[[노트 제목]]이라고 쓰면 다른 노트로 연결돼요. 이 링크가 쌓이면서 노트들이 서로 이어진 지식 네트워크가 만들어져요.

        

        

          
🕸️

          
그래프 뷰

          
노트(점)와 링크(선)를 한눈에 보여주는 지도예요. 위의 아이디어 그래프가 바로 이 그래프 뷰의 축소판이에요.

        

      


      
자주 묻는 질문

      

        

          여기 적은 노트는 어디에 저장되나요?
          
내 브라우저의 localStorage에만 저장돼요. 서버에는 저장하지 않으며, AI 분석 요청 시에만 노트 내용이 전송돼요. 브라우저 데이터를 지우면 노트도 사라지니 중요한 내용은 복사해두세요.

        

        

          옵시디언은 유료인가요?
          
개인 사용은 무료예요. 기기 간 동기화(Obsidian Sync)나 웹 게시(Publish) 같은 부가 서비스만 유료예요.

        

        

          AI가 찾은 연결을 옵시디언으로 어떻게 가져가나요?
          
분석 결과의 "옵시디언용 마크다운 복사" 버튼을 누르면 [[내부링크]]와 태그가 포함된 마크다운이 복사돼요. 옵시디언 새 노트에 붙여넣으면 링크가 바로 인식돼요.

        

        

          노트는 몇 개까지 분석할 수 있나요?
          
한 번에 2개 이상 20개 이하의 노트를 분석할 수 있어요. 노트 하나당 내용은 500자까지 적을 수 있어요.

        

        

          챗봇이 옵시디언과 관계없는 질문에도 답하나요?
          
이 챗봇은 옵시디언 전문 도우미라서, 무관한 질문은 정중히 사양하고 옵시디언 관련 질문을 안내해드려요.
